<h2>멘토링 4 : GPT</h2>
(정기영 멘토님)

1. Intro<br><br>
: : 모든 [자연어] task가 가능하다 -> <u>모든 [설명 가능한] task가 가능하다!</u>
: : ex) GPT로 체스 두기 (체스는 자연어 태스크는 아니지만, 설명함으로써 가능하게 되었다)
: : ex) GPT를 이용한 추리 게임
: : ex) 주가 예측 - 주가 데이터 다 넣어주고 맞추게 하는 것
<br><br>
: : 즉, GPT에게 "설명"해줘야 한다
: = 좋은 "Prompting" 이 필요하다

<h2>2. Prompting</h2>
- What is prompting?<br><br>
: : 모델들에게 입력되는 지시문이나 질문
: : 모델이 어떻게 반응하고, 어떤 종류의 정보나 내용을 생성할지를 결정하는 데 중요한 역할을 한다
<br><br>
: 1) 명확한 지시
: 2) 예상 결과의 방향 제시
: 3) 상황 설정
<br><br>
: <br><b>프롬프팅 꿀팁 1 - Few-shot</b><br><br>
: : Zero-shot은 예시 없는 것, One-shot은 예시가 하나인 것.
: : Few-shot은 몇 개의 예시를 주는 것
: : 엣지 케이스들(아웃라이어, 특이한 케이스들) 뽑아서 3,4개 이상, 많아야 10~20개 정도 제공해주면 Few-shot의 성능이 잘 나온다고 한다.
<br><br>
: : '페르소나'에서는 특정 상황에 대한 리액션, 자주 하는 말투, 대략적인 답변의 길이, 등을 퓨샷으로 조절 가능
: : ex. 말투 (~~하냐, ~ 하냐구 등)
: : ex. 대답만 하지 않고 끝에 질문을 되묻는다든지 등..
<br><br>
: <br><b>프롬프팅 꿀팁 2 - Chain of Thought</b><br><br>
: : 아웃풋에서 결과로 바로 가는 것이 아니라, 중간에 한 번 내용을 정리하고, 생각하라고 지시하는 것
: : ex. 챗봇에서는 상대가 입력한 대화에 답변을 바로 생성하는 것이 아니라, 의도를 파악하거나 감정을 파악하거나 어떤 식으로 대응할지 한 번 고민하고, 답변을 생성할 수 있음.
<br><br>
: <br><b>프롬프팅 꿀팁 3 - Step by Step</b><br><br>
: : Pause Token을 써서 생각할 시간을 만들어주면? 생성 능력이 더 좋아진다. Let's Think Step by Step
: <br><b>프롬프트를 구매/판매</b><br><br>
: : PromptBase라는 Prompt 판매 사이트도 있다.
: : 프롬프트 엔지니어로서의 포트폴리오를 쌓고 싶다면, 이런 곳에 올려서 팔아도 좋은 포트폴리오가 되는 것 같다.
<br><br>
: <br><b>프롬프트 인젝션</b><br><br>
: : 프롬프트 중간에 굉장히 강력한 input을 하나 넣어주는 것.
: : 시스템의 응답을 조작하거나, 원래 의도와 다른 방향으로 유도할 수 있다.
: : GPT를 중심으로 동작하는 서비스에서 욕/혐오발은 등이 나오게 할 수도 있다.
: : GPT가 막아 놓은 주가 예측, 가치 판단 등의 기능을 활용해 볼 수도 있다.
: : GPT로 서비스를 하면, 이런 부분이 가장 큰 문제가 된다.

<h2>4. Q&A</h2>
: <br><b>질문 답변</b><br><br>
: : **Q**) 정교한 기능이 필요한 태스크를 맡길 수 있나?
: : GPT 3.5는 점점 성능이 안좋아지고 있다.
: : GPT 4를 사용해보면 아마 다 해결될 것 같다.
: : Lang chain처럼 GPT를 여러 레이어로 쓰는 건데,
: : GPT에게 Query를 생성하라고 하고, 또 한번 GPT에게 물어보는 것 (오류가 있는지 수정하도록 하고)
: : GPT를 Loop 안에서 계속적으로 다른 서비스와 소통하면서 할 수 있도록 만드는 방법도 있음.
: : Auto GPT라고, GPT에게 그런것들을 결정하게 만드는 것도 있음. (레이어 쌓고 결과 확인 등등.. 프롬프팅도 스스로 하며 무한 루프를 돌리게 하는 것)
: : 한글 프롬프팅은 한계가 있다! 입력, 출력이 다 한글이어도 프롬프팅은 영어로 하는 것이 훨씬 더 성능이 좋다.
<br><br>
: : Llama 등 모델을 사용하면, 내 마음대로 처음부터 끝까지 다 프롬프팅을 할 수 있다~ (GPT 3.5 ~ GPT 4 사이 정도의 성능은 나오는 듯)
